Projeto pronto para o desafio "Desenvolvimento Front-end II".
Arquivos:
- index.html
- form.html
- confirmation.html
- style.css
- script.js
- data.json (exemplo)

Fluxo:
1. Abrir form.html e preencher.
2. Enviar leva para confirmation.html.
3. Confirmar gera download de data.json e redireciona para index.html.

Observação:
Browser não permite escrever diretamente arquivos no servidor — por isso a confirmação gera um download do arquivo data.json (com o mesmo efeito de salvar localmente).
